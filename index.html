Welcome to OmniPhaser's Agar.io Modscripts repository!

You can install all of the latest and greatest mods for Agar.io from this repository easily, and we edit the open-source scripts to add additional functionality!

Installation Instructions

1. Install Userscript Extension

If you are on Google Chrome/Chromium, install the Tampermonkey extension from the Chrome Web Store. This will allow you to install userscript mods for Agar.io.

If you are on Firefox, install Greasemonkey from the Add-on store.

2. Install scripts

Install each script listed below by clicking on the link and pressing the "Raw" button on the top right corner of the editor that appears. This should open a link to Tampermonkey, select the Install button to install the script. Repeat this for each script listed below.

  • Mod loader
  • Mod launcher
  • Bot logic script
  • Ph4nt0m_pwn Agariomods.com Connect script
    • /

3. Play modded Agar.io

Now you can head over to Agar.io and enjoy the awesome new mods you have installed! Thank you for trying the scripts! Be sure to check for updates, and if you have any questions, post them on issues on the GitHub project page.
